Le site Travel Agency

Nouvel exercice HTML et CSS

Toujours un site statique avec HTML et CSS

Après le site du photographe fictif Robbie Lens d'OpenClassrooms, cette fois ci, j'ai dans l'idée de me lancer dans une nouvelle session d'apprentissage. S'il est plutôt bien fait, ce site ne m'auras pas beaucoup plus apporté que le premier.

Comme pour le site précédent, le but, ici,est d'apprendre les bases du HTML et CSS. Cette fois ci, c'est avec la plateforme "Apprendre-a-coder.com" que je suivrais ce cours, découverte sur le site Udemy

Même s'il présente plutôt bien, le site "Travel Agency" reste néanmoins très basique. Le site n'est composé que d'une page.

Mise en page des textes, introduction des lien, attributs, positionnement, styles, couleur, tableaux, formulaires, flexbox, systèmes de grid, responsive, l'exercice est assez complet.
