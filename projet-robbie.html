Le site de Robbie Lens

Première expérience HTML et CSS

Création de site statique avec HTML et CSS

Le site du photographe fictif Robbie Lens est certainement le premier site de nombreux développeurs, comme ce fut le cas pour moi. En effet, il s'agit du cours HTML du célèbre site d'apprentissage "OpenClassrooms".

L'idée, ici, est de nous faire découvrir en douceur les bases du HTML et CSS. Accessible, ce cours nous fais découvrir les bases du développement front-end. Le cours est dispensé par Mathieu Nebra lui-même, fondateur du site.

Le site de Robbie Lens nous permet de découvrir les points clés du HTML. S'il nous appartient de les approfondir, un large éventail des possibilités offertes par le HTML et le CSS nous sont ici dévoilés.

Mise en page des textes, introduction des lien, attributs, positionnement, styles, couleur, tableaux, formulaires, flexbox, systèmes de grid, responsive, l'exercice est assez complet.
